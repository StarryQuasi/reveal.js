設計模式運用

  • 觀察者模式 (Observer)

    可綁定屬性於物件屬性改變時自動同步其他綁定的物件

  • 建造者模式 (Builder)

    優化方塊定義流程，只留必要參數，美化程式

  • 組合模式 (Composite)

    使用樹狀結構構建模組化 GUI

觀察者模式 (Observer)

Bindable<T> 於物件屬性改變時自動同步其他綁定的物件


Bindable<int> p1;
Bindable<int> p2;
p1.onValueChanged([](const auto& from, const auto& to) {
    std::println("p1 {} -> {}", from, to); });
p2.onValueChanged([](const auto& from, const auto& to) {
    std::println("p2 {} -> {}", from, to); });
p2.bindTo(p1);
p1.set(1);
// p1 0 -> 1
// p2 0 -> 1
p2.set(2);
// p2 1 -> 2
// p1 1 -> 2
std::println("p1 is {}", p1.get()); // p1 is 2
std::println("p2 is {}", p2.get()); // p2 is 2

建造者模式 (Builder)

BlockBuilder 整合流式接口 (Fluent interface) 優化方塊定義流程，只留必要參數，美化程式


                const Block Blocks::copperOre = builder()
                    .name("Copper Ore")
                    .item(Items::copperOre)
                    .loot().min(0).item(Items::cobblestone) // Default of max is 1
                    .loot().max(3).item(Items::copperIngot) // Default of min is 1
                    .build();
                const Block Blocks::cherryLog = builder()
                    .name("Cherry Log")
                    .item(Items::cherryLog)
                    .build();
                const Leaves Blocks::cherryLeaves = builder()
                    .name("Cherry Leaves")
                    .item(Items::cherryLeaves)
                    .transparent(true)
                    .build<Leaves>();
                const CraftingTable Blocks::craftingTable = builder()
                    .name("Crafting Table")
                    .item(Items::craftingTable)
                    .build<CraftingTable>();

a.k.a. C++ 版 HTML


Hotbar::Hotbar(Props props) :
    FlowContainer(props)
{
    for (int i = 0; i < slots.size(); i++) {
        Container* cont = this->addChild<Container>({.size = {{20, 20}}});
        selectors[i] = cont->addChild<Container>({
            .size = {{24, 24}},          // 大小
            .anchor = Anchor::midMiddle, // 置中
            .origin = Anchor::midMiddle,
            .visible = false,            // 隱藏
        });
        slots[i] = cont->addChild<Slot>();
    }
    selectors[0]->setVisible(true);
}
